Модель Користувачів та Пристроїв

Структура "Користувач -> Багато ПК -> Багато Записів"

Оновлена Схема БД (PostgreSQL)

Users

  • id (PK)
  • email, password
  • role (admin / user)

PCs

  • id (PK)
  • user_id (FK)
  • pc_identifier (HWID / Name)
  • last_seen_at

Activity (Processes)

  • id (PK)
  • pc_id (FK)
  • process_name, title
  • start_time, duration

Процес Авторизації

Web (Blade)

Стандартна сесія Laravel. Користувач бачить тільки свої ПК. Адмін бачить список усіх користувачів та їхні пристрої.

Desktop App (API)

Надсилає дані разом з токеном користувача. Якщо `pc_identifier` новий — сервер автоматично створює запис у таблиці PCs для цього користувача.
