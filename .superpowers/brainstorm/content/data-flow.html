База даних (PostgreSQL)

PCs Table

Зберігає назву ПК та API Key. Зв'язаний з таблицею Admins (користувачами веб-панелі).

Processes Table

Використовує UUID як первинний ключ (для уникнення конфліктів при синхронізації з різних пристроїв).

Docker Compose Інфраструктура

  • app: PHP 8.3 FPM (Laravel)
  • web: Nginx
  • db: PostgreSQL 16
  • network: logger-network
