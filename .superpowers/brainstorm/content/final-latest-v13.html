Фінальна Специфікація (Laravel 13 + PostgreSQL 18)

Використання найновіших версій ПЗ

Оновлений Технологічний Стек

  • Framework: Laravel 13.x
  • Database: PostgreSQL 18
  • PHP: 8.4+
  • Authentication: JWT (JSON Web Token)
  • Containerization: Docker Compose (Apache + Postgres 18)
Деталізація API v1

Автентифікація

  • POST /api/v1/auth/login — Логін (email/password) -> JWT
  • POST /api/v1/auth/register — Створення акаунта
  • POST /api/v1/auth/refresh — Оновлення токена

Синхронізація (Batch)

  • POST /api/v1/sync/processes — Масив активностей вікон + pc_unique_id
  • POST /api/v1/sync/schedules — Масив статусів ПК + pc_unique_id

Дані та ПК

  • GET /api/v1/pc/list — Перелік пристроїв поточного користувача
  • GET /api/v1/pc/{id}/activity — Перегляд історії за конкретним ПК

Архітектурні рішення

  • Service Layer: Бізнес-логіка винесена в сервіси для чистоти контролерів.
  • Auto-PC Creation: Сервер сам створює ПК в базі при отриманні нового ID.
  • Blade Integration: Веб-інтерфейс використовує ті ж сервіси, що і API.
