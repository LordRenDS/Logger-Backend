Специфікація API та Технології

Використання Laravel 12.x та PostgreSQL 17

Технологічний Стек (Latest)

  • Framework: Laravel 12.x (остання версія)
  • Database: PostgreSQL 17 (остання стабільна версія)
  • Auth: JWT (php-open-source-saver/jwt-auth)
  • Docker: php:8.4-apache & postgres:17
Заплановані API Ендпоінти (v1)
Метод Шлях Опис
POST /api/v1/auth/login Вхід, повертає JWT токен
POST /api/v1/auth/register Реєстрація нового користувача
POST /api/v1/auth/logout Вихід (інвалідація токена)
GET /api/v1/auth/me Інформація про поточного юзера
POST /api/v1/sync/processes Batch Sync: Відправка масиву активностей вікон
POST /api/v1/sync/schedules Batch Sync: Відправка статусів ПК (on/off)
GET /api/v1/pc/list Список пристроїв користувача

Особливості реалізації

Кожен запит на /sync/* буде містити pc_unique_id. Якщо сервер не знаходить такого ПК у базі — він автоматично створює новий запис у таблиці PCs та прив'язує його до авторизованого користувача.
